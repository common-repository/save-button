== Description ==

Adds a Save button to post editing page, which saves the post and redirects back to the posts list.

**FEATURES**

- Save button on post editing page.
- Keeps all post filters on redirect.
- No configuration needed.

More information can be found on <a href="https://thekrotek.com/wordpress-extensions/miscellaneous">extension's page</a>.

== Installation ==

1. Unpack and upload all files to "/wp-content/plugins/plugin-name" directory. Or simply install plugin through Plugins page.
2. Activate extension on "Plugins" page.
3. Look for "Save" or "Publish" button, when editing or creating a post.
